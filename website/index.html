everything you'd open five apps for

One board. Every kind of page you actually need.

Corkboards

Pin sticky notes, reference palettes, and mechanic cards anywhere on a zoomable canvas. Lock, rotate, and resize — it stays exactly how you left it.

Kanban

Custom columns, drag-and-drop cards, and a full detail panel per card — description, checklist, image references. Sprint boards without the SaaS bill.

Spreadsheets

A real formula engine — refs, ranges, SUM(), arithmetic — for scope budgets and drop tables, resizable well past a fixed grid.

Draw

Freehand sketching for level layouts and quick mockups, on the same zoomable, resizable canvas as your boards.

Code + Run

An editor for HTML/CSS/JS with a real Run button — preview a prototype or a devlog page without leaving the app.

Game-dev module pack

GDD outlines, an asset pipeline tracker, bug reports with severity, and a dated playtest log — built in, not bolted on.

Home dashboard

A widget-based overview, not another blank page — drop in a file list, a note, or a live feed of what's flagged and due.

Reminders + Calendar

Flag any card or note with "remind me on Home," add a due date, and see everything overdue or upcoming on a real month grid.

Everything is a plugin

Boards, views, and dashboard widgets all register through the same open contract — the built-ins use nothing a community add-on couldn't.

home dashboard

Your project's front page, built out of the same widgets community plugins use.

Home isn't a file — it's a real section with its own grid layout. Pin a Quick Jump list, a freeform note, or the Reminders widget that pulls in every flagged card and module across the whole project, soonest due date first.

  • Click a reminder to jump straight to the file it lives in
  • Overdue items are called out in red automatically
  • New widget types register the same way a new module type does
